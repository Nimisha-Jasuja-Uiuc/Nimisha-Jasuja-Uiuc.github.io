Name of the dataset is “P1-UK-Bank-Customers”. I found the dataset on the below web page under module 6, Table calculations, more Dashboard and Storytelling. I found it quite useful when it comes to creating dashboard

https://www.superdatascience.com/pages/tableau. The dataset represents information about customers of a bank located in UK. The dataset represents generic details of the customers of the bank. There are in total of 4014 rows and 9 columns. The list of fields are Customer ID’, ‘Name’, ‘Surname’, ‘Gender’, ‘Age’, ‘Region’, ‘Job Classification’, ‘Date Joined’, ‘Balance’. Each row represents a customer. I used Idyll’s built in component table to display the dataset. This componnent was very useful since the dataset was extremely large and the ‘table’ component provided an interactive way and easy way to view the table at a glance which was otherwise difficult.
